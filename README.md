# DoorAccessControl
门禁管理系统已更新到4.1

基于RFID的门禁系统
    程序流程：
    识别卡片ID，判断是否为管理员卡片，
    是管理员ID，则再次读卡，
    如果为管理员卡片，则退出
    如果为已知ID则从存储器删除，为未知ID则写入存储器钟中；
    如果第一次读卡不是管理员ID，
    若为已知ID，则开门，若为为止ID，则不允许通过；

    修复了1.0版本中的以下问题：
    [1] 无法将门禁信息写入EEPROM中
    [2] 完善继电器选择模式
    [3] 解决运行过程中，擦除按键无法清拆主卡信息的问题
    [4] 解决除主卡外，录入的第一门禁卡无法识别的问题

    在2.0版本基础上增加远程控制

    在3.3版本基础上增加柜子灯光控制
